# Chrome Web Store Listing - ko

## Name
Theater Mode Everywhere - 시어터 모드

## Short summary
키 하나로 모든 HTML5 비디오 플레이어를 브라우저 화면 전체에 확대합니다.

## Full description
Theater Mode Everywhere는 키 하나만 누르면 모든 HTML5 비디오 플레이어를 브라우저 화면 전체로 확대하여, 깔끔하고 몰입감 있는 시네마틱 환경을 제공합니다.

영상을 볼 때마다 헤더, 사이드바, 댓글, 추천 영상 등으로 가득한 화면에 지치셨나요? 튜토리얼, 라이브 방송, 짧은 클립 등 어떤 영상이든 키 하나로 즉시 몰입할 수 있는 시어터 환경으로 전환할 수 있습니다.

주요 기능
- 원키 몰입 모드: `T`를 눌러 활성 비디오를 브라우저 전체 영역으로 확대합니다. 다시 `T` 또는 `Escape`를 누르면 원래 화면으로 돌아갑니다. 페이지에 여러 비디오가 있으면 `Shift+T`로 전환할 수 있습니다.
- 세련된 컨트롤 패널: 음량 및 재생 속도를 정밀하게 조절할 수 있는 세로 팝업 메뉴, 반투명 유리 효과(glassmorphic) 툴팁, 편리한 시간 표시기가 포함된 현대적인 사용자 인터페이스를 제공합니다.
- 음량 부스트 (최대 300%): Web Audio API를 활용하여 소리가 작은 영상의 음량을 최대 300%까지 증폭할 수 있으며, 부드러운 비선형 슬라이더로 정밀하게 조절하고 설정 메뉴에서 활성화할 수 있습니다.
- 화면 표시(HUD): 음량 변경, 탐색, 재생 전환 시 아름다운 애니메이션 피드백을 제공합니다.
- Shadow DOM 지원: 최신 웹 기술인 Shadow DOM 내부에 캡슐화된 비디오 플레이어도 찾아내어 확대할 수 있어, 다양한 최신 웹사이트에서 안정적으로 작동합니다.
- 사용자 정의 단축키: 모드 활성화, 탐색, 프레임 단위 이동 등을 위한 키 조합을 원하는 대로 설정할 수 있습니다. 내장 도움말이 현재 활성 단축키를 항상 알려줍니다.
- 예외 목록: 특정 웹사이트에서 확장 프로그램 작동을 차단하여 원본 디자인과의 충돌을 방지할 수 있습니다. 브라우저 도구 모음 팝업 또는 설정 페이지에서 관리할 수 있습니다.

지원 및 제한 사항
- 범용 HTML5 비디오: Shadow DOM 내부의 플레이어를 포함하여 모든 웹사이트의 표준 `<video>` 요소에서 작동합니다.
- 임베디드 플레이어 (iFrame): YouTube나 Vimeo 등에서 프레임을 통해 삽입된 비디오를 인식하고 확대합니다.
- 비표준 자막: HTML5 내장 자막(`<track>` 태그)은 완벽하게 지원됩니다. 사이트가 커스텀 레이어나 자체 JS 스크립트를 통해 자막을 표시하는 경우, 커스텀 컨트롤 패널에 표시되지 않을 수 있습니다.
- DRM 보호: 일부 스트리밍 플랫폼은 DOM 내 비디오 객체 조작을 금지하는 저작권 보호 기술(DRM)을 사용하여 플레이어 확대가 제한될 수 있습니다.

시작하기
1. 영상이 있는 웹페이지를 열어주세요 (예: 영상 블로그, 뉴스, 스트리밍).
2. `T` 키를 누릅니다 (커서가 입력 필드나 댓글란에 있지 않은지 확인하세요).
3. 하단 커스텀 패널을 사용하고, `←` / `→` 화살표 키로 탐색하며, `<` / `>` 키로 프레임 단위 이동을 합니다.
4. `Escape` 또는 `T`를 눌러 시어터 모드를 종료합니다.
