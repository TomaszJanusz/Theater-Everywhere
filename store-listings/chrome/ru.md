# Chrome Web Store Listing - ru

## Name
Theater Mode Everywhere - Режим кино

## Short summary
Разверните любой видеоплеер HTML5 на всю область просмотра браузера одним нажатием клавиши.

## Full description
Theater Mode Everywhere разворачивает любой видеоплеер HTML5 на все окно браузера одним нажатием клавиши, обеспечивая чистый кинематографический просмотр без отвлекающих факторов.

Theater Mode Everywhere создан для всех, кто хочет наслаждаться веб-видео без нагромождения шапок, боковых панелей, комментариев или рекомендаций. Независимо от того, смотрите ли вы учебные руководства, стримы или короткие ролики, вы можете превратить страницу с любым видео в уютный кинотеатр одним нажатием клавиши.

ОСНОВНЫЕ ФУНКЦИИ
- Иммерсивный режим одной клавиши: нажмите `T`, чтобы мгновенно расширить активное видео под размеры окна браузера. Нажмите ее снова или `Escape`, чтобы вернуться в обычный режим. Если на странице несколько видео, нажмите `Shift+T`, чтобы переключаться между ними.
- Элегантное управление плеером: чистая современная панель управления с вертикальными всплывающими ползунками громкости и скорости воспроизведения, полупрозрачными (glassmorphic) подсказками и индикаторами оставшегося/прошедшего времени.
- Усиление звука (до 300%): увеличивайте громкость тихого видео до 300% с помощью Web Audio API, используя нелинейный ползунок для точной настройки и переключатель в настройках.
- Визуальный отклик на экране (HUD): получайте красивые анимированные индикаторы на экране при регулировке громкости, перемотке или переключении воспроизведения/паузы.
- Поддержка Shadow DOM: находит и разворачивает видеоплееры, скрытые внутри теневых корней (Shadow DOM) современных веб-компонентов, что улучшает совместимость с большим количеством сайтов.
- Настраиваемые горячие клавиши и параметры: настройте собственные клавиши для переключения режима, перемотки и покадрового просмотра. Информационный оверлей помощи в самом плеере позволяет легко их вспомнить.
- Список исключений: исключайте определенные домены через всплывающее окно панели инструментов браузера или страницу настроек, чтобы предотвратить конфликты с собственным дизайном сайтов.

ЧТО ПОДДЕРЖИВАЕТСЯ
- Универсальное HTML5 видео: работает на стандартных элементах HTML5 `<video>` по всему интернету, включая плееры, инкапсулированные внутри Shadow DOM.
- Встроенные плееры (iFrames): обнаруживает и разворачивает видеоэлементы, встроенные во фреймы, такие как плееры YouTube или Vimeo.
- Нестандартные субтитры: поддерживаются стандартные субтитры HTML5 (теги `<track>`). Если веб-сайт отображает субтитры через пользовательские слои, собственный JavaScript или внешние элементы, эти субтитры могут не отображаться внутри панели управления расширения.
- Сервисы с защитой DRM: некоторые стриминговые платформы используют технологии DRM, ограничивающие манипуляции с DOM видео, что может помешать расширению правильно масштабировать видео.

КАК НАЧАТЬ
1. Откройте любую страницу с видео, например видеохостинг, новостной сайт или блог.
2. Нажмите клавишу `T` на клавиатуре. Убедитесь, что курсор не находится в строке поиска или поле для комментариев.
3. Используйте элементы управления внизу плеера, стрелки `Влево` / `Вправо` для перемотки и `<` / `>` для покадрового перехода.
4. Нажмите `Escape` или `T`, чтобы выйти из режима кино.
