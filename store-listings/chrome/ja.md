# Chrome Web Store Listing - ja

## Name
Theater Mode Everywhere - シアターモード

## Short summary
キーを1回押すだけで、HTML5ビデオプレーヤーをブラウザの表示領域いっぱいに拡大します。

## Full description
Theater Mode Everywhereは、キーを1回押すだけで、あらゆるHTML5ビデオプレーヤーをブラウザウィンドウ全体に表示し、余計な表示のない、動画に集中できる快適なシネマ体験を提供します。

ヘッダー、サイドバー、コメント、おすすめ動画などの邪魔な要素なしで、ウェブ動画を楽しみたいすべての人のために設計されています。チュートリアル、配信、短いクリップなどを視聴する際、キーを押すだけで、あらゆる動画ページを心地よいシアターに変えることができます。

主な機能
- ワンキー・イマーシブモード: `T`キーを押すだけで、再生中の動画をブラウザの表示領域いっぱいに瞬時に拡大します。もう一度押すか、`Escape`キーを押すと、元のサイズに戻ります。ページ内に複数の動画がある場合は、`Shift+T`キーで動画を順番に切り替えることができます。
- 洗練されたカスタムコントロールバー: 音量や再生速度調整用のポップアップパネル、グラスモーフィズムデザインのツールチップ、残り時間/経過時間のインジケーターを備えた、クリーンでモダンな操作パネルです。
- 音量ブースト機能（最大300%）: Web Audio APIを使用して、音量が小さい動画の音声を最大300%まで増幅します。精密な調整が可能な非線形スライダーと、設定での有効/無効切り替えに対応しています。
- HUDでの視覚的フィードバック: 音量調整、シーク、再生/一時停止の切り替え時に、画面上にエレガントなアニメーションインジケーターを表示します。
- Shadow DOMのサポート: モダンなシャドウルート（Shadow DOM）内にカプセル化されて隠れているビデオプレーヤーも検出して拡大できるため、多くのウェブサイトで動作します。
- カスタムショートカットと詳細設定: シアターモードの有効化、シーク、フレーム送りのキー設定を好みに合わせてカスタマイズできます。プレーヤー内のヘルプメニューからいつでも簡単に確認できます。
- 除外リスト機能: ブラウザのツールバーポップアップや設定ページから、特定のドメインを除外リストに登録して、サイト本来のレイアウトと干渉するのを防ぐことができます。

対応環境と制限事項
- 汎用HTML5ビデオ: ウェブ上の標準的なHTML5 `<video>` 要素で動作します。Shadow DOM内に配置されたプレーヤーも含まれます。
- 埋め込みプレーヤー（iFrame）: YouTubeやVimeoの埋め込みコードなど、フレーム内に配置された動画を自動的に検出して拡大します。
- 独自仕様の字幕について: 標準のHTML5字幕（`<track>` タグ）に対応しています。ただし、サイト独自のJavaScriptや外部読み込みなどで独自に重ねて表示している字幕は、シアターモードのカスタムコントロール内では正しく表示されない場合があります。
- DRMで保護された配信サイト: 一部の動画配信サイトでは、動画DOMの操作を制限する著作権保護技術（DRM）が使用されているため、動画の拡大スケールが正しく行われない場合があります。

使い方
1. 動画共有サイト、ニュースサイト、ブログなど、動画が表示されているページを開きます。
2. キーボードの `T` を押します（検索ボックスやコメント欄に入力中ではないことを確認してください）。
3. 画面下部に表示されるカスタムコントロール、`Left` / `Right` 矢印キーでのシーク、`<` / `>` キーでの1フレームずつの送り操作などを利用できます。
4. `Escape` または `T` を押すと、元のサイズに戻ります。
