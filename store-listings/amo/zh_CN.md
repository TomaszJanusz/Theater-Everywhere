# AMO Listing - zh_CN

## Name
Theater Mode Everywhere - 剧场模式

## Summary
只需按一个键，即可让任何 HTML5 视频播放器填满浏览器视口。

## Description
Theater Mode Everywhere 只需一个按键，就能让任何 HTML5 视频播放器填满整个浏览器窗口，带来干净、无干扰的影院式观看体验。

Theater Mode Everywhere 面向所有想要摆脱页眉、侧边栏、评论和推荐内容干扰的网页视频用户。无论你在观看教程、直播还是短视频，都可以用一个快捷键把任意视频页面变成舒适的剧场。

主要功能
- 一键沉浸模式：按 `T` 立即将当前视频扩展到浏览器视口。再次按下或按 `Escape` 即可恢复。如果页面上有多个视频，可按 `Shift+T` 在它们之间切换。
- 精致的自定义播放器控件：现代化控制栏，包含音量和播放速度的垂直弹出面板、玻璃拟态提示，以及剩余/已播放时间显示。
- 音量增强（最高 300%）：使用 Web Audio API 将较小的视频音量增强到最高 300%，配合非线性滑块进行精细调节，并可在设置中开关。
- 可视 HUD 反馈：调节音量、跳转进度或切换播放/暂停时显示优雅的动画叠层。
- Shadow DOM 支持：遍历并放大隐藏在现代 shadow roots 中的视频播放器，提高更多网站的兼容性。
- 自定义快捷键和设置：为切换、跳转和逐帧播放配置自己的按键。播放器内帮助叠层便于查看。
- 排除列表：通过浏览器工具栏弹窗或选项页排除特定域名，避免与网站原生布局冲突。

支持范围
- 通用 HTML5 视频：适用于网页上的标准 HTML5 `<video>` 元素，包括封装在 Shadow DOM 内的播放器。
- 嵌入式播放器（iFrames）：检测并放大嵌入在 frame 中的视频元素，例如 YouTube 或 Vimeo 嵌入播放器。
- 非标准字幕：支持标准 HTML5 字幕（`<track>` 标签）。如果网站通过自定义叠层、专有 JavaScript 或额外挂载元素渲染字幕，这些字幕可能不会显示在自定义叠层中。
- DRM 保护服务：部分流媒体平台使用 DRM 技术限制视频 DOM 操作，可能导致扩展无法正确缩放视频。

如何开始
1. 打开任何包含视频的页面，例如视频网站、新闻网站或博客。
2. 按键盘上的 `T`。请确保你当前没有在搜索框或评论框中输入文字。
3. 使用底部自定义控件，按 `Left` / `Right` 箭头跳转，按 `<` / `>` 逐帧播放。
4. 按 `Escape` 或 `T` 退出剧场模式。
