# AMO Listing - uk

## Name
Theater Mode Everywhere - Режим кінотеатру

## Summary
Максимізуйте будь-який HTML5-відеопрогравач на весь екран браузера одним натисканням клавіші.

## Description
Theater Mode Everywhere розгортає будь-який HTML5-відеопрогравач на всю ширину та висоту вікна браузера всього одним натисканням клавіші, створюючи чистий кінематографічний простір без зайвих деталей.

Це розширення створене для тих, хто втомився від нагромадження заголовків, бічних панелей, коментарів та рекомендацій під час перегляду відео. Незалежно від того, чи дивитеся ви навчальні посібники, стріми чи короткі кліпи, ви можете миттєво перетворити будь-яку відеосторінку на затишний кінотеатр одним натисканням клавіші.

ОСНОВНІ ХАРАКТЕРИСТИКИ
- Режим занурення однією клавішею: Натисніть `T` для швидкого розгортання активного відео на всю область браузера. Натисніть `T` ще раз або `Escape` для повернення до звичайного режиму. Якщо на сторінці кілька відео, натисніть `Shift+T` для перемикання між ними.
- Елегантна панель керування: Сучасний інтерфейс користувача з вертикальними спливаючими меню для точного налаштування гучності та швидкості відтворення, напівпрозорими підказками (ефект матового скла - glassmorphic) та зручними індикаторами часу.
- Посилення звуку (До 300%): Збільшуйте гучність тихого аудіо в розгорнутих відео до 300% за допомогою Web Audio API, з плавним нелінійним повзунком для точного регулювання та можливістю активації в меню налаштувань.
- Візуальний відгук на екрані (HUD): Отримуйте гарні анімовані підказки під час зміни рівня гучності, перемотування або перемикання відтворення.
- Сумісність із Shadow DOM: Знаходить та розгортає навіть приховані відеоплеєри, інкапсульовані всередині сучасних тіньових коренів (Shadow DOM), що забезпечує стабільну роботу на багатьох сучасних сайтах.
- Власні гарячі клавіші: Налаштуйте зручні для вас комбінації клавіш для активації режиму, перемотування чи покадрового перегляду. Вбудована довідка завжди нагадає активні клавіші.
- Список виключень: Заблокуйте роботу розширення на певних вебсайтах через спливаюче вікно на панелі інструментів браузера або сторінку налаштувань, щоб уникнути конфліктів з оригінальним дизайном.

ПІДТРИМКА ТА ОБМЕЖЕННЯ
- Універсальне HTML5 відео: Працює зі стандартними елементами `<video>` на будь-якому вебсайті, включаючи плеєри всередині Shadow DOM.
- Вбудовані плеєри (iFrame): Розпізнає та розширює відео, вбудовані через фрейми, наприклад, з YouTube чи Vimeo.
- Нестандартні субтитри: Повністю підтримуються вбудовані субтитри HTML5 (теги `<track>`). Якщо сайт відображає субтитри через кастомні шари або власні скрипти JS, вони можуть не відображатися на кастомній панелі керування.
- Захист DRM: Деякі стрімінгові платформи використовують технології захисту авторських прав (DRM), які забороняють маніпуляції з відеооб'єктами в DOM, що може заважати коректному розгортанню плеєра.

ЯК ПОЧАТИ
1. Відкрийте будь-яку сторінку з відеороликом (наприклад, відеоблог, новини чи стримінг).
2. Натисніть клавішу `T` (переконайтеся, що курсор не знаходиться у полі введення чи коментарях).
3. Використовуйте кастомну панель знизу, стрілки `Вліво` / `Вправо` для перемотування та клавіші `<` / `>` для покадрового кроку.
4. Натисніть `Escape` або `T` для виходу з режиму кінотеатру.
